<p>
  This simple iOS app converts entered temperature into Farenheit or Celcius depending on user's selection. This app makes use of the following:
</p>
<ul>
  <li>
    The view controller layout which contains two Labels, one button, and one textfield
  </li>
  <img src="https://github.com/angelineflorajohn/iOS-apps/assets/25217498/8d380f09-0cdc-4774-926b-2c9ae1beb7b7" alt="homeScreen">
  <li>
    One TextField will allow entering the temperature
  </li>
  <li>
    The "Convert to F" button will convert the entered temperature to Farenheit and display the converted temperature in one of the labels
  </li>
  <img src="https://github.com/angelineflorajohn/iOS-apps/assets/25217498/b7e3cb89-76dc-4c36-9fd3-a35f11fe70b7" alt="temperatureInFarenheit">
  <li>
    The "Convert to C" button will convert the entered temperature to Celcius and display the converted temperature in one of the labels
  </li>
  <img src="https://github.com/angelineflorajohn/iOS-apps/assets/25217498/57157291-fbd3-4e7e-aac3-1d338945a19d" alt="temperatureInCelcius">
  <li>
    The second label located on the top left corner displays the number of successful temperature conversions
  </li>
</ul>
